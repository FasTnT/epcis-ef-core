Client Library

Another project was developed to make the EPCIS subscription callback much easier for the developers.

This library allows to register callback endpoints and parses the subscription content to a usable list of event classes. It support both .NET Minimal Api and ASP.NET MVC controllers endpoints.

Like the repository, it is hosted in github: https://github.com/louisaxel-ambroise/epcis-callback.
